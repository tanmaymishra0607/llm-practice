# GenAl Streamlit Frontend

A simple Streamlit frontend for the GenAl project.

## Run the app

```bash
streamlit run streamlit_app.py
```

## Features

- Text analysis with character, word, and sentence counts
- Top word frequency display
- Sample blood work report rendering
- Basic abnormal value highlighting for the sample report

## Notes

- This is a demo frontend, not a medical diagnostic tool.
- The sample report is loaded from `2_health_analysis/blood_work.txt`.
